# DrawCanvas

DrawCanvas is a mobile drawing app built for Android using React Native and Expo(SDK 54). It turns your mobile screen into a digital drawing pad where you can sketch, paint, or take notes simply by touching the screen.

## Main Features

### Custom Brushes: 
You can change your brush color and adjust how thick or thin your lines are using a smooth slider. It also has an Eraser Mode to easily wipe away mistakes.

### Undo and Redo:
The app remembers every single stroke you make. If you make a mistake, you can press Undo to go back. If you change your mind, you can press Redo to bring it back.

### Save:
The app uses AsyncStorage to save your drawings to your phone's memory. Even if you close the app or turn off your phone, your artwork will not be deleted, and you can open it again later.

### What Tools Were Used?

React Native & Expo: 
The main framework used to build the app so it runs smoothly on both Android and iPhones.

TypeScript: 
A tool that helps write clean code and stops programming errors before the app even runs.

Zustand: 
A lightweight tool that manages all the drawing logic, like keeping track of your undo/redo history and your layers.

AsyncStorage: 
A built-in mobile database used to save your drawing files directly onto your phone.
